# Bitmap
Take home technical test


## Run This Project Locally

Clone the project

```bash
  gh repo clone git@github.com:sebnaylor/Bitmap.git
```

Go to the project directory

```bash
  cd Bitmap
```

Install dependencies

```bash
  bundle install
```

Start the server

```bash
  ruby lib/bitmap_interface.rb
```

Run rspec test

```bash
  bin/rspec
```
